About the Project

This python program uses OpenCV and qr code to create and recognize codes


Application create QR code for each student
Student have to scan the QR code before entering the class

When student scan his code.Student name and data will be added to attendence file

There by at the end of the day teacher can find the total attendence

Note : Only authorized QR codes are alloweded to be added in attendence file, all other will be discarded

For GUI we uses tkinter 



Main FIle : scanner.py 


mydatafile.txt contains the username who are authorised.This data will added when you give add to data base button in QR code generation window

attendence.txt contains the attendence: Only Authorized user name and data will show here.
